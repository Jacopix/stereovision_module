Information regarding compatible_cameras.json


Source: 
    https://developer.nvidia.com/embedded/jetson-partner-supported-cameras


File structure
    all_cameras
        All cameras compatible with Jetson Nano and MIPI interface

    available_cameras (subgroup of all_cameras)
        All cameras for which a vendor was found

    lowlight_cameras (subgroup of available_cameras)
        All cameras with "low-light" feature
